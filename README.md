# Ambulance Intelligente pour la Priorisation des Patients

## Description
Ce projet devlopper par AHARRAN Noura, AMEKRAN Ouadia, AKHRIF Salma et BEN AYAD Douaae, vise à développer un système intelligent pour la **priorisation des patients** dans les ambulances, en utilisant des techniques d’**Intelligence Artificielle** et de traitement de données.  
L’objectif est de permettre aux équipes médicales de gérer efficacement les urgences en **évaluant automatiquement la gravité des cas** et en optimisant le parcours des ambulances.
